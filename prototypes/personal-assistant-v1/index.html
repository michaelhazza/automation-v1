Personal Assistant V1 — Round 1 Prototypes
Hi-fi clickable prototypes for the three new user-facing surfaces in EA V1. Every other UI surface in the build (Connections row, Recurring Tasks, Triggers, Run Trace) reuses an existing visual pattern and is not mocked here.
EA V1 brief → Foundation brief →
Foundation predecessor: these mockups assume principal-scoped-agents has merged. The EA is a user-principal agent — each user has their own instance bound to their identity, with credentials and memory scoped to them. Org admins see metadata only; content is redacted by default.

Reused without new mockups
No new visual design for:

  • System-agent template: shipped via DB seed migration like Sarah / Johnny / Helena / Patel / Riley / Dana.

Review questions to look for as you click through
  • 01: Is the connect / questionnaire / voice-profile sequence the right order? Does the privacy framing on each step feel sufficient?
  • 02: Does the Personal nav group sit correctly above Operate / Build / Govern? Does the home view feel useful at 7am on a Tuesday or is it just a list of cards?
  • 03: Is the settings page comprehensive without being overwhelming? Should anything move into a "advanced" section?
