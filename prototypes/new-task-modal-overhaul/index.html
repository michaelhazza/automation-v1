New Task Modal Overhaul
Enriched NewTaskModal replacing today's minimal NewBriefModal. Three screens showing the three key states of the same modal. Extends client/src/components/layout/modals/NewBriefModal.tsx.
01
Default state
Modal just opened. Title empty. Instructions empty. Agent picker on default. Drop-zone empty with fallback button visible. Advanced section collapsed. Primary action: Create Task (disabled until title is filled).
→
02
With attachments (mid-upload)
Title and Instructions filled. Agent selected. One file fully uploaded, one file uploading (animated progress bar). Lifecycle notice explains that task creation and uploads are independent operations. Create Task enabled.
→
03
Advanced expander open (full complexity)
All fields visible: Title, Instructions, Agent, Attachments (1 file), Due Date, Priority, Subaccount override, Org override (with admin-only pill). Demonstrates the maximum-field state while remaining scannable.
→
