These five pages cover the places where the UX isn't obvious and where the payload shapes need to stand up to real screens before routes lock in. If the information hierarchy is wrong here, the payload shape is wrong. Skip pack CRUD, document CRUD, permission settings — standard shapes, borrow from existing memory_blocks / agents pages.
