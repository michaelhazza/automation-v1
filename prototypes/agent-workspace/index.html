Phase 2: Agent Workspace (post Phase 1 split)

Mockups for review

Five mockups demonstrating the agent workspace embodiment surface remaining on this branch after the auto-knowledge-retrieval feature was split out as Phase 1. Brief: docs/agent-workspace-implementation-brief.md. Strategic spine: docs/agent-cloud-compute-dev-brief.md (Rev 5, locked).

Reading order: 1 (Home, workspace-level presence) → 2 (Agent Overview active, the centerpiece) → 3 (Overview idle) → 4 (Overview first-run, identity instantiation) → 5 (Run trace with file lineage).
MOCKUP 1
Home: Active Agents widget enhancement
home-active-agents.html
Replaces the current single-number Active Agents tile with a live status list per running agent (current focus in plain language, elapsed time, pulsing dot) plus scheduled-next agents below. Workspace-level presence surface.
MOCKUP 2
Agent Overview: active state
agent-overview-active.html
New default landing tab on Agent edit. Agent is currently working. Hero shows status pill, current focus, elapsed time. Recent observations, recent activity feed, files snapshot, knowledge in use, schedule peek, connections health, performance. The centerpiece of the brief.
MOCKUP 3
Agent Overview: idle state
agent-overview-idle.html
Same shell, agent is not currently running. Hero shifts to last-seen and next-scheduled. Activity feed prominent. Active goal callout makes the surface feel alive even when nothing is executing.
MOCKUP 4
Agent Overview: first-run state
agent-overview-first-run.html
Brand-new agent with no run history. Welcome banner, three quick actions, setup checklist, identity card, and empty states with helpful copy. The agent feels like an entity that already exists, not a placeholder. Identity instantiation demo per the strategic spine.
MOCKUP 5
Run trace: inline file lineage
run-trace-lineage.html
Existing Run trace surface gets file chips inline on each step that produced a file. Click the chip to deep-link into the file in Knowledge: Files (Phase 1 surface). Small visual change, big workflow win.
