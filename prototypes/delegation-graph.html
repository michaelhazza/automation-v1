Why the graph shape matters

Every edge records delegation_direction on the child run — down for normal descent, up for escalation to the parent, lateral for root-only cross-subtree handoff. The direction is immutable once written, so delegation patterns are explainable in one pass without joining through mutable task state.

  • •Rejected attempts show in-line so the user understands why a branch exists without a subtree under it.
  • •The opened run is highlighted; double-click any node to pivot the view to that run's trace.
  • •Graph is reconstructed from agent_runs alone — no joins through tasks.
