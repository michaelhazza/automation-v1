Locked architectural decisions surfaced in UI
Duration cap
120 min per run, operator-tier override allowed (up to 240 min)
Concurrency cap
3 active sessions per subaccount
Crash policy
Fail the run (no auto-restart)
Artefact harvest
End of session + on-demand snapshot
Sandbox persistence
Persistent across turns (one sandbox per session)
Image versioning
Pinned per session (in-flight runs complete on their original image)
