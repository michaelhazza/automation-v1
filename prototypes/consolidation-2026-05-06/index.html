Confirmed decisions (not up for re-debate)

  • Knowledge = Memory (auto) + Authored knowledge. Review Queue moved to Inbox.
  • Skills drop from main nav. Surfaced inside Agent Edit (Capabilities tab) as kit+library pattern.
  • Inbox unifies all HITL surfaces: tasks, approvals, belief conflicts, block proposals, clarifications, failed runs.
  • Automation Detail is NOT a top-level nav item. Opens from automation name row click.
  • Connections organises by service (one row per service), not by credential type.
  • Categories renamed to Tags in all UI copy.
  • OrgSettingsPage Memory and Integrations tabs removed from Manage Org. Both have dedicated nav items.
  • Workflows and Automations stay separate. Workflow steps can call automations.
  • Client Pulse deferred. Not in this consolidation.
