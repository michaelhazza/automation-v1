Prototype Round 1

Synthetos UI Consolidation

A major UX consolidation pass reducing ~99 pages and ~45 left-nav items into a coherent, task-first navigation structure. Each mockup is rooted in the existing source pages to guarantee no fields are silently dropped.

Date: 2026-05-06
Round: 1 of N
Status: Awaiting operator review
Files: 10 screens

Confirmed decisions (not up for re-debate)

  • Knowledge = Memory (auto) + Authored knowledge. Documents stay separate.
  • Skills drop from main nav. Surfaced inside Agent edit only.
  • Workflows and Automations stay separate. Workflow steps can call automations.
  • Briefs removed entirely (handled in a separate cleanup PR).
  • Categories renamed to Tags in all UI copy.
  • Health Findings renamed to Issues. Moved into Integrations as an Issues tab.
  • Connectors (ClientPulse data ingestion) and Integrations (MCP/agent tools) stay separate. Only Integrations gets consolidation.
