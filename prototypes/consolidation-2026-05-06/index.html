UI Consolidation Prototype

Automation OS — UI Consolidation

Clickable HTML prototypes for the consolidated UI. Each card links to a redesigned surface; if a "View before" link is present, it opens the depiction of the current production state for comparison. Use the demo profile selector at the top of the sidebar (Workspace operator / Org admin / System admin) to see permission-gated views.

Date: 2026-05-06
Status: Awaiting operator review

Confirmed product decisions

  • Three sidebar modes (Workspace / Org / System) with permission-gated mode switcher.
  • Inbox unifies all human-in-the-loop surfaces: approvals, clarifications, tasks, failed runs, memory updates.
  • Knowledge has three tabs: Authored memory, Auto-memory, Documents (with Documents and Bundles sub-tabs).
  • Memory is split by who wrote it (you vs the system). Pending review is a status that applies in either tab.
  • Org-inherited content is hidden from workspace operators. Visible to org admins as "Admin view" with a tooltip.
  • Capabilities tab retired. Replaced by separate Skills and Data sources top-level tabs on agent edit. Skills tab uses search-driven UX scaling to 200+ skills.
  • MCP tool servers are invisible to operators. Skills are the unit of agent capability; MCP-derived skills are bundled by capability theme (Option C).
  • Triggers retired as a standalone surface. Folded into a unified "Recurring tasks" page with Schedule / Event / Manual fire conditions.
  • Workflows + External automations consolidated under one Automations page with tabs.
  • Goals retired. Replaced by a per-project Objective field that is read by agent execution at runtime.
  • ClientPulse stays as a separate thread (links in sidebar, surface untouched).
  • Spend and cap widget on home; full Spending page consolidates Budgets and Ledger.
  • Connections page uses plain language: Sign in / Key / Web login replace OAuth / API key / web_login.
  • Onboarding uses multi-select for integrations / agents / automations; background data pull begins on each connection.
  • Calendar nav: Today + previous / next inline with the period label; view switcher (Week / Fortnight / Month) on the right.
