SkillStudioPage and SkillAnalyzerPage are admin-only tools accessible separately from agent management. Operators configuring an agent must navigate: AdminAgentEditPage > Capabilities tab > edit link > AdminSkillEditPage. SkillStudio adds a third context-switch for custom skill creation. In the consolidated view, all skill management lives inside the agent's Capabilities tab as a drawer.
