Organisation model (by cred type vs service)

The consolidated Connections page organises by service (one row per service) rather than by credential type. Each row shows all credential types for that service, a health badge inline, and opens a drawer with credential detail, agent tool access, data sync settings, issues tab, and audit log. Multiple credential types (API key, OAuth, Web login, Cookie session, Certificate) are supported per service.
