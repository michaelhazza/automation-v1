Current state, Workflows (2 pages, one access-controlled)

Workflows is a separate primitive from Automations. WorkflowsLibraryPage is the operator-facing template browser and run-launcher. WorkflowStudioPage is a system-admin authoring tool. WorkflowRunDetailPage (subaccount sub-directory) shows individual run status. All three are currently navigable by any authenticated user if they know the URL.
