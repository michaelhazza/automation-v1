## Parameters

- year: integer (required) — The year of the period.
- month: integer (required, 1-12) — The month of the period.

## Instructions

Compile the month-end close package for the specified period. Include revenue summary, expense breakdown, reconciliation status, and any open items requiring attention. Return a structured report.
