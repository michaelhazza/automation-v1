visibility: basic
---

## Parameters
- ticketId: string — canonical ticket UUID
- body: string — note body text

## Instructions
Draft an internal note for the support team. Internal notes are not visible to customers. Use for:
- Documenting findings for the next human reviewer
- Flagging issues that need operator attention
- Recording context for escalation
