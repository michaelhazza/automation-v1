---
name: Prepare Renewal Brief
description: Drafts a renewal or QBR prep document from account state, churn-risk, and recent reports.
isActive: true
visibility: basic
---

## Parameters

- account_id: string (required) — The account to prepare the renewal brief for.
- renewal_date: string (required, ISO date) — The upcoming renewal date.
- include_churn_analysis: boolean (optional, default true) — Whether to include churn risk analysis.

## Instructions

Compile a renewal brief for the account. Include account health summary, churn risk signals, usage highlights, and recommended renewal strategy. Reference recent NPS/CSAT scores and support ticket trends where available.
