Organisation Templates

Platform-wide library of organisation archetypes. Each template ships an agent hierarchy blueprint + operational defaults (scoring weights, alert limits, intervention templates, etc.). Orgs adopt one of these on signup; the org-level operational config layered on top is owned by the org-admin and edited via ClientPulse Settings or the Configuration Assistant.

🧭

How this differs from Subaccount Blueprints

Organisation Templates (this page) define the archetype of an entire organisation — the agent hierarchy, the seeded operational defaults (weights, bands, intervention templates), the required integrations. System-admin only.
Subaccount Blueprints (org-admin facing) define how a per-client subaccount inside an org should be set up. Org admins can author their own blueprints and pick a default for new subaccount creation.

GHL Agency Intelligence

Reference template for GoHighLevel agencies. Ships ClientPulse health-score + churn-risk scoring, intervention defaults, staff-activity classifier, integration-fingerprint scanner, and the full agent hierarchy (Portfolio Health Agent + sub-agents).
