# openclaw-session template

Placeholder scaffolding. Real implementation lands with the OpenClaw adapter spec; V1 CI does not build, scan, or publish this template. The `CURRENT_VERSION` file carries a single-line placeholder (`version=0.0.0-placeholder`) rather than the five-field production contract — once the OpenClaw adapter spec activates this directory, that spec will add the remaining four required fields and extend the `verify-template-version-coherence` gate to include this path.
