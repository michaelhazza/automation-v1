Self-diagnosis test: each error block answers three questions — what broke, where, and what to do next. The "Fix" line + the primary action button are the things that prevent a support ticket.
Pre-dispatch vs post-dispatch: the failure panel's tracing block explicitly shows the dispatch event was NOT emitted (matches §5.9 contract). Operators looking at logs get the same signal.
Input preservation on failure: showing the rendered input that would-have-been-sent is key for debugging template-expression bugs ({{ steps.X.output.Y }}).
