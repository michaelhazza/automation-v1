First-class vs bolt-on: "Call Automation" sits in the same menu as Invoke Skill and Agent Decision — not in a secondary "advanced" submenu. The ribbon highlight + "New" badge earn attention on launch; post-launch it becomes a neutral sibling.
External-call signal: the amber bolt icon visually differentiates Automation steps from the indigo/violet native-step palette. Users can scan a Workflow canvas and count external dependencies at a glance.
Explore-Mode preview chip: "pauses in Explore Mode" is shown in the menu row so the user sets expectations before picking the step type. Avoids surprise at run time.
